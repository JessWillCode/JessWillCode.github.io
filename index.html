CSS Selectors & Specificity
Cheat Sheet

Basic Selectors

Selector Example Description
Universal (*) * { margin: 0; } Targets all elements.
Element (Type) p { color: red; } Targets all <p> elements.
Class (.) .box { padding: 10px; } Targets elements with class="box".
ID (#) #main { width: 100%; } Targets an element with id="main".

Grouping & Combining Selectors

Selector Example Description
Grouping (A, B) h1, h2, p { color: blue; } Targets multiple elements.
Descendant (A B) div p { font-size: 16px; } Targets <p> inside <div>.
Child (A > B) nav > ul { list-style: none; } Targets direct children only.
Adjacent Sibling (A + B) h1 + p { margin-top: 10px; } Targets the first <p> right after <h1>.
General Sibling (A ~ B) h1 ~ p { color: gray; } Targets all <p> siblings after <h1>.

Attribute Selectors

Selector Example Description
Exists ([attr]) [disabled] { opacity: 0.5; } Targets elements with the attribute.
Exact Match ([attr="value"]) [type="text"] { color: blue; } Targets exact attribute values.
Contains ([attr*="value"]) [class*="btn"] { padding: 5px; } Targets attributes containing "btn".
Starts With ([attr^="value"]) [href^="https"] { color: green; } Targets attributes starting with "https".
Ends With ([attr$="value"]) [src$=".jpg"] { border: 1px solid; } Targets attributes ending with ".jpg".

Pseudo-Classes

Selector Example Description
:hover a:hover { color: red; } When hovered over.
:focus input:focus { border: 2px solid blue; } When focused (e.g., input field).
:first-child p:first-child { font-weight: bold; } First child of parent.
:last-child p:last-child { font-style: italic; } Last child of parent.
:nth-child(n) li:nth-child(2) { color: red; } Targets the nth child.
:nth-of-type(n) p:nth-of-type(2) { color: red; } Targets the nth of its type.
:not(selector) button:not(.disabled) { background: blue; } Excludes elements that match the selector.

Pseudo-Elements

Selector Example Description
::before p::before { content: "★"; } Adds content before an element.
::after p::after { content: "✔"; } Adds content after an element.
::first-letter p::first-letter { font-size: 2rem; } Styles the first letter of a paragraph.
::first-line p::first-line { font-weight: bold; } Styles the first line of a paragraph.
::selection ::selection { background: yellow; } Styles text when highlighted.

Advanced (Combinators & Special Selectors)

Selector Example Description
:root :root { --main-color: blue; } Targets the root (<html>) for CSS variables.
:empty div:empty { display: none; } Targets elements with no content.
:checked input:checked { background: green; } Targets checked checkboxes/radios.
:disabled button:disabled { opacity: 0.5; } Targets disabled elements.

CSS Specificity & Importance

  • Inline styles (e.g., <p style="color:red;">) have the highest priority.
  • ID selectors (#id) are stronger than class selectors (.class).
  • Using !important (e.g., color: red !important;) overrides other rules.
